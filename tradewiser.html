Summary

Tradewiser is an app that helps to manage account holder's trading and financial assets. The application uses both livestock & currency data from Yahoo Finance and ExchangeRate-API sites respectively, to help user with trading and banking. Ultimately, the app enables user to deposit and withdraw funds, to buy and sell stock, and to analyse these assets.

The idea for this project was inspired by the Finance application in week 9, which I found to be lacking many features, an interest in stock trading, and a lingering desire to create a banking application. As you can imagine, this being my first full-on application, I have kept on having more and more idea, and therefore kept adding features. And thus, there are still many features that I intend to add, such a schedule reporting, and logging, or transaction COMMIT/ROLL BACK, etc. Due to personal deadlines, I have resolved that these additional features will be added later.
